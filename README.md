# 🏦 Loan Default Predictor with AI Advisor

An interactive Streamlit web application that predicts the probability of loan default using Machine Learning and provides AI-powered financial advisory using Groq LLMs.

---

## 🚀 Features

- Logistic Regression prediction model
- SMOTE handling for extreme class imbalance
- Real-time probability prediction
- AI Loan Advisor powered by Groq
- Interactive Streamlit interface
- Mathematical breakdown of predictions
- Train/Test ROC-AUC diagnostics

---

## 🧠 Technologies Used

- Python
- Streamlit
- Scikit-learn
- SMOTE (imbalanced-learn)
- Groq API
- Pandas
- NumPy

---

## 🌐 Live Website

[Visit the Website Here](https://carborrowing.streamlit.app/)

---

## 📊 Model Information

The application:

- Trains a Logistic Regression model
- Uses SMOTE for class balancing
- Evaluates using ROC-AUC
- Predicts loan default probability
- Provides AI-generated financial recommendations
